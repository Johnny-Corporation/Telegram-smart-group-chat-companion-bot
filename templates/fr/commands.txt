Je suis JohnnyBot. Je peux discuter, faire appel, répondre, suggérer des solutions à vos messages.
Si vous êtes nouveau sur JohnnyBot, veuillez consulter la /description

Vous pouvez me contrôler en envoyant ces commandes :

<b>Fonctions de configuration</b>
 /change_langage - change la langue du bot

 /set_temperature - définit le chaotique dans les réponses
 /temporary_memory_size - définit la valeur maximale des messages mémorables


<b>Fonctions d'interaction</b>
 /question_to_bot - pose une question sortie de son contexte

 /enable - activer l'automodé. Le robot lit les messages et décide quand il doit répondre aux messages
 /disable - active le mode automatique

 /enable_dialog - active le mode dialogue. C'est un chat, un bot répond à chacun de vos messages
 /disable_dialog - désactive le mode dialogue.


<b>Personnalisation</b>
 /dynamic_generation - active la génération dynamique. Le robot imprime le message devant vos yeux


<b>INFOS</b>
 /description - voir la description du bot
 /commands - liste de toutes les commandes avec explication


<b>Paramètres du compte</b>
 /messages_info - combien de messages vous avez utilisés
 /about - message de démarrage


<b>Assistance</b>
 /report_bug - envoyer aux développeurs des informations sur le bug
 /request_feature - suggère une nouvelle fonctionnalité du bot aux développeurs
